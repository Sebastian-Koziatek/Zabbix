### Dodawanie Zabbix do Grafany

1. Wykonaj polecenie za pomocą którego poprzez CLI Grafany ściągniesz potrzebny plugin. Następnie wykonaj restart serwisu.
```bash
sudo grafana-cli plugins install alexanderzobnin-zabbix-app
sudo systemctl restart grafana-server
```

2. Wejdź w sekcję Administration i wybierz opcję plugins.
3. Wyszukaj plugin **"zabbix"**
![Zabbix](/grafiki/grafana_1.png)

4. Wejdź w niego i użyj opcji "Enable"
![Zabbix](/grafiki/grafana_1.png)

5. Wejdź teraz w **Connections > Data sources** i znajdź tam Zabbix
   ![Zabbix](/grafiki/grafana_3.png)
   6. W url wprowadź adres IP serwera Zabbix z końcówką /zabbix/api_jsonrpc.php, np. http://zabbix.sadmin.pl/zabbix/api_jsonrpc.php
   7. Wprowadź użytkownika i hasło:
  ![Zabbix](/grafiki/grafana_4.png)
  8. Kliknij save & test na dole, powinno pojawić się potwierdzenie z aktualną wersją
  ![Zabbix](/grafiki/grafana_5.png)
